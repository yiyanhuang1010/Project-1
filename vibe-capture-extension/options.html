API settings

The side panel can call Google Gemini (generativelanguage.googleapis.com) or Anthropic Claude (api.anthropic.com) with your screenshot. Pick the model in the sidebar, then paste the matching key below.

Create a key in Google AI Studio. Used when a Gemini model is selected.

Create a key in Anthropic Console. Used when a Claude model is selected.

Keys are stored only in Chrome sync storage on your device; they are not sent to the extension author.
